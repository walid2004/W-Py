# W-Py
A full fledged machine learning &amp; math library

A from-scratch Python library for machine learning, data structures, and algorithms. Implements core ML concepts and classical algorithms without relying on external ML frameworks.

## Overview

W-Py provides implementations of machine learning algorithms, graph algorithms, and data structures built from first principles. Similar in scope to scikit-learn and NumPy, but designed for understanding fundamentals and educational purposes.

## Features

Machine Learning - Regression algorithms including linear regression, polynomial regression, and regularized variants. Classification algorithms including logistic regression, decision trees, KNN, SVM, and Naive Bayes.

Data Structures and Algorithms - Sorting algorithms including bubble sort, merge sort, quicksort, heap sort, and insertion sort. Graph algorithms including shortest path algorithms, depth-first search, breadth-first search, and minimum spanning trees.

Utilities - Data preprocessing and normalization. Train-test data splitting. Performance metrics for regression and classification.

## Installation

Clone the repository and install in development mode.

## Project Structure

wpy/ contains classification, regression, graphs, algorithms, preprocessing, and metrics modules. tests/ directory for unit tests. examples/ directory for usage examples. docs/ directory for detailed documentation.

## Comparison with Other Libraries

W-Py differs from scikit-learn and NumPy in that it is built from scratch for learning purposes, focuses on algorithm clarity and understanding, includes graph algorithms not typically found in ML libraries, and prioritizes educational value over production performance.

W-Py is suited for learning, understanding algorithms, small-scale projects, and educational purposes. Scikit-learn and NumPy are recommended for production systems, performance-critical applications, and complex workflows.

## Requirements

Python 3.8 or higher. No external ML dependencies required.
